# 🌍 Flag Quiz

A simple and interactive flag quiz built with **HTML**, **CSS**, and **Vanilla JavaScript**.

The application fetches country data from the REST Countries API and challenges users to identify flags through multiple-choice questions.

## 🚀 Features

- Random country flags
- Multiple choice questions
- Countdown before quiz starts
- Score tracking
- Timer
- Final results screen
- Responsive design

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)
- REST Countries API
